# Samsung-300K-Tool
A tool that forces the galaxy devices to reboot in download mode

# How to use?

Download [Samsung 300K Tool](https://github.com/av2xn/Samsung-300K-Tool/releases/download/v1.0/Samsung-300K-Tool.exe) and run it from cmd like this:

```bash
Samsung-300K-Tool.exe 1
```

Or

```bash
Samsung-300K-Tool.exe 2
```
